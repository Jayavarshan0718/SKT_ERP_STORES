# SKT_ERP_STORES

An Enterprise Resource Planning (ERP) Store Management System developed using **ASP.NET Web Forms**, **C#**, and **SQL Server** to automate inventory and store operations.

---

## Overview

SKT_ERP_STORES is a web-based ERP application designed to simplify and manage daily store activities such as inventory management, gate pass processing, purchase requests, stock tracking, and reporting. The system helps organizations maintain accurate inventory records, improve operational efficiency, and reduce manual paperwork.

---

## Features

- User Authentication and Role-Based Access
- Inventory and Stock Management
- Gate Pass Management
- Purchase Request Management
- Goods Receipt Management
- Goods Issue Management
- Supplier Management
- Dashboard and Reports
- Search and Filter Records
- Crystal Reports Integration
- Real-Time Inventory Tracking

## Installation

1. Clone the repository.

```bash
git clone https://github.com/yourusername/SKT_ERP_STORES.git
```

2. Open the project in **Visual Studio 2022**.

3. Restore NuGet packages.

4. Configure the SQL Server connection string in **Web.config**.

5. Build and run the application.
